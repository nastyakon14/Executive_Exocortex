Разработка интеллектуальной системы персонального знания и принятия решений «Экзокортекс топ-менеджера»  (AI-based Executive Exocortex: Voice Knowledge Graph and Semantic Reasoning System)

"Современные руководители работают с огромным объёмом неструктурированной информации — голосовые заметки, письма, чаты, документы, совещания. Потеря контекста и сложности извлечения нужной мысли создают когнитивное «узкое горлышко» в управлении.
Проект направлен на разработку цифрового экзокортекса — системы, способной автоматически собирать и структурировать знания руководителя в виде семантической карты его “мыслей”.
Система объединяет распознавание речи, парсинг сообщений, векторизацию содержания, построение графа смысловых связей и RAG-бота для диалогового доступа к знанию.
Это фактически реализация цифрового Zettelkasten, встроенного в экосистему корпоративного общения."
